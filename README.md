# tokotura

Snake guide.

## Installation

```sh
pip install git+https://github.com/noipeK/tokotura.git
```

There may be some optional requirements, you can install them all with all.

```sh
pip install -U git+https://github.com/noipeK/tokotura.git#egg=tokotura[all]
```
